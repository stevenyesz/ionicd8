ionic + drupal8 移动开发开源示例项目

IONIC 是目前最有潜力的一款 HTML5 手机应用开发框架。通过 SASS 构建应用程序，它提供了很多 UI 组件来帮助开发者开发强大的应用。 它使用 JavaScript MVVM 框架和 AngularJS 来增强应用。提供数据的双向绑定，使用它成为 Web 和移动开发者的共同选择。Ionic是一个可以创建类似于手机平台原生应用的一个开发框架，可以实现编译成各个平台的应用程序。Ionic以流行的原生移动开发SDK为蓝本，使开发过原生iOS或安卓app的任何人都容易理解。开始只需书写你的代码，完成时通过PhoneGap/cordova发布，Ionic让你看不出混合应用和原生的区别，一次开发，处处运行。

Drupal8 是最新推出的一款PHP-based后端开发框架，3290为贡献者，1228家公司数年合力开发，专为移动而生，天然支持RESTful 接口。RESTful简单来说就是为后端业务提供了一套API，任何形式的前端，包括ios,android,微信，PC 都可以通过这套API 获取数据，提交业务。目前Drupal社区已经有多大3万多个功能模块，前面覆盖包含电商，支付，社交，物流，问卷调查等等几乎你能想到的所有业务，为前端提供强力支持。
